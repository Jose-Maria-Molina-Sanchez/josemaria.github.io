Mis proyectos

Son dos los proyectos que realizado hasta ahora.

El primero es un formulario completo que está diseñado de manera que desde ningún campo se pueda inyectar código.

El segundo es una aplicación de escritorio desarrollada en C# y Windows Forms para la gestión de clientes y pedidos de una base de datos Oracle, como parte de un proyecto de clase. El proyecto demuestra la implementación de una arquitectura limpia y profesional utilizando el patrón de diseño DAO (Data Access Object).
